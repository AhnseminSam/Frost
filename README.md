# Frost
A simple chatbot that can chat with users and solve basic math problems. It also allows users to switch between chat mode and math mode.
Math mode, it can calculate the expressions entered by the user. It only allows numbers and basic math operators and you can go to chat mode by typing 'exit'.
Chat mode, it can chat with the user and respond to some basic questions. It also allows users to switch to math mode by typing 'math_mode' or exit the chatbot by typing 'exit'.
It can only answer a few questions for now.
